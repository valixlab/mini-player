# Mini player pour spotify

Le player de spotify est trop gros, et le rapetisser est une feature premium. Alors je l'ai rendu _gratuit_.

✅ Simple d'installation et d'utilisation
✅ Léger (lightweight)
✅ Fonctionne sur n'importe quel système APT

🎵 Le player récupère automatiquement votre flow de musique depuis _playerctl_ et vous l'affiche joliement. Il reste discrêtement en haut de toutes vos fenêtre pour quand même pouvoir y accéder quand vous jouer ou vous travailler.

![Capture d'écran du mini-player](https://i.imgur.com/fUsFzvJ.png)

# INSTALLATION

📦 1) Téléchargez le .zip, dé-zipper. C'est dans ce dossier que sera enregistrer toutes les données, placer le dans un endroit sûr (par exemple le dossier "~/")

⚙️ 2) Lancez le script d'installation :

``bash setup.sh``

🔓 3) Un raccourci est placé sur votre bureau. Faites un clic-droit dessus puis "autoriser l'exécution".

🥳 Votre mini player est prêt !

📝 -> L'application enregistre automatique là où vous la positionner et la couleur de thème choisi.
🎧🟢 (la couleur de thème par défaut est le vert de spotify)
