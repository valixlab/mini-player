# Mini player pour spotify

Le player de spotify est trop gros, et le rapetisser est une feature premium. Alors je l'ai rendu _gratuit_.

✅ Simple d'installation et d'utilisation

✅ Léger (lightweight)

✅ Fonctionne sur n'importe quel système APT

🎵 Le player récupère automatiquement votre flow de musique depuis _playerctl_ et vous l'affiche joliement. Il reste discrètement en haut de toutes vos fenêtres pour quand même pouvoir y accéder quand vous jouez ou vous travaillez.

![Capture d'écran du mini-player](https://i.imgur.com/fUsFzvJ.png)

# INSTALLATION

📦 1) Téléchargez le .zip puis dézippez-le. C'est dans ce dossier que seront enregistrées toutes les données, placez-le dans un endroit sûr (par exemple le dossier "~/")

⚙️ 2) Lancez le script d'installation :

```bash
bash setup.sh
```

🔓 3) Un raccourci est placé sur votre bureau. Faites un clic-droit dessus puis autorisez l'exécution.

🥳 Votre mini player est prêt !

📝 -> Le player enregistre automatiquement là où vous le positionnez et la couleur de thème choisie.
🎧🟢 (la couleur de thème par défaut est le vert de spotify)
